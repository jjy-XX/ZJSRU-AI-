# 智慧校园交互系统

> 本项目是浙江树人学院《人工智能》课程大作业

一个基于AI大模型的智慧校园交互式系统，包含校园导览、精细化课程生成、场所预约和数字人助手四大核心功能。

---

## 项目简介

智慧校园交互系统是一个基于Web的交互式应用，旨在通过人工智能技术提升校园服务体验。系统集成了大语言模型API、虚拟数字人、AIGC图像生成等前沿AI技术，为师生提供智能化的校园服务。

项目完全在浏览器中运行，无需安装任何客户端，采用纯前端技术实现，具有轻量、跨平台、易部署等特点。

## 功能介绍

### 🔐 登录系统
- 登录验证界面，含学号/工号和密码
- 登录状态本地持久化存储
- 用户信息显示与退出登录
- 演示账号：student / 123456、admin / admin、teacher / 123456
- 登录后方可使用四大功能模块

### 🏠 首页
- 系统概览与功能导航
- 精美的功能卡片，各配真实图片
- 核心功能快速入口
- 点击卡片直接跳转对应模块

### 🗺️ 校园虚拟导览
**核心功能：沉浸式校园探索**

- 🏫 6大校园景点：校门、图书馆、教学楼、宿舍、食堂、运动场
- ▶️ 自动导览模式
- 📖 景点详细介绍
- 📊 景点数据展示
- 🎨 精美的视觉效果

### 📅 精细化课程生成
**核心功能：AI文本生成/数据分析 · 院系-专业-课程三级架构**

- 🏛️ **9大学院**：信息科技学院、管理学院、人文与外国语学院、艺术设计学院、医学院、城建学院、继续教育学院、生环学院、工程学院
- 📚 **32个专业**：计算机科学与技术、软件工程、人工智能、数据科学与大数据技术、数字媒体技术、工商管理、会计学、金融学、市场营销、国际经济与贸易、英语、汉语言文学、汉语国际教育、日语、法学、视觉传达设计、数字媒体艺术、环境设计、产品设计、动画、临床医学、护理学、土木工程、建筑学、成人高等教育、职业培训、风景园林、环境工程、生物工程、机械设计制造及其自动化、电子信息工程
- 🎓 **4个年级**完整课程体系
- 👨‍🏫 每门课程含教师、教室、学分、学时、课程描述
- 📋 课程列表查看与统计
- 📥 课程表下载功能
- 🔄 一键重新生成

### 📝 场所预约系统
**核心功能：校园资源时段预约管理**

- 🚌 **班车预约**：3条线路，6个班次，上下行双向
- 📖 **图书馆预约**：4个自习室，7个时段
- 🏃 **体育馆预约**：篮球场、羽毛球场、乒乓球室、健身房
- 📌 **会议室预约**：4个会议室，8个时段
- 📅 日期前后切换选择
- ✅ 实时显示预约人数与容量
- 📋 我的预约列表
- ❌ 支持取消预约
- 💾 本地存储预约记录

### 🤖 数字人虚拟助手
**核心功能：虚拟数字人实时交互**

- 👤 纯CSS绘制的数字人形象
- 👄 口型同步动画
- 😊 多种表情与心情系统
- 🔊 语音合成播报
- 👀 眼球追踪（跟随鼠标移动）
- 💃 肢体动作动画
- 💬 实时对话交互

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构 |
| CSS3 | 样式与动画（含数字人绘制） |
| JavaScript (ES6+) | 交互逻辑 |
| AI大模型API | 数字人对话（支持OpenAI/DeepSeek/通义千问） |
| Web Speech API | 语音合成 |
| LocalStorage | 本地数据存储（预约记录） |
| AIGC图像生成 | 图像资产（Midjourney / DALL·E） |

## 快速开始

### 方式一：直接打开
直接双击 `index.html` 文件即可在浏览器中运行。

### 方式二：本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .
```

然后访问 `http://localhost:8000`
